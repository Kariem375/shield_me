# Shield Me - Classic Final Demo

نسخة محافظة على التصميم الكلاسيكي للمشروع:

- الصفحة الرئيسية عبارة عن الشات فقط.
- فحص ملفات وروابط من نفس صندوق الشات.
- صفحة "من نحن" بنظام كروت توضح خدمات المنصة.
- سجل الفحص يظهر للمستخدمين المسجلين فقط، وكل حساب له سجله الخاص على نفس الجهاز.
- شاشة الانتقال بين الصفحات في منتصف الشاشة كما كانت في التصميم الأصلي.
- الواجهة متجاوبة مع الكمبيوتر والموبايل.

## التشغيل المحلي

افتح Terminal داخل فولدر المشروع ثم شغل Backend:

```powershell
py -m pip install -r requirements.txt
py app.py
```

اترك هذه النافذة مفتوحة.

افتح Terminal جديد داخل نفس الفولدر وشغل Frontend:

```powershell
py -m http.server 5500
```

افتح الموقع من:

```text
http://127.0.0.1:5500
```

لا تفتح `index.html` مباشرة من الجهاز.

## اختبار Backend

افتح:

```text
http://127.0.0.1:5000/health
```

إذا ظهرت استجابة JSON فهذا يعني أن خدمة الفحص تعمل.

## ملاحظة

الفحص متاح بدون تسجيل دخول، لكن سجل الفحص لا يتم حفظه أو عرضه إلا بعد تسجيل الدخول.
